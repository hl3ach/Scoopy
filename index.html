Scoopy the Forklift Robot

ECE 118 Final Project – Group 2: Andrew Huang, Hayden Leach, Anthony Furman

Project Overview

Scoopy is an autonomous forklift-style robot designed to navigate a reconfigurable grid, locate and lift DC motors, and deposit them at a target location, all in under 1 minute. This project demonstrates integration of mechanical, electrical, and software subsystems to meet dynamic environmental constraints.

Subsystem Design

Mechanical System

Scoopy’s mechanical system features a belt-driven elevator mounted on vertical rails. The scoop is attached to a carriage that rides this elevator, with two servo motors controlling its tilt angle. The elevator belt is driven from the bottom via a secondary belt connected to a DC motor powered through an H-bridge. The drivetrain consists of two front-mounted DC motors with directly attached wheels, and two passive ball-bearing skegs at the rear for support and stability. All CAD for the mechanical components was created in Onshape and fabricated using laser-cut wood and Bambu Lab A1 Mini 3D prints.

Electrical System

Scoopy runs on three separate 9.9V LiFePO4 batteries. One powers the Uno32 control stack, onboard sensors (through 3.3V regulators), and servo motors (through 5V regulators). The second battery supplies all four DC motors — two for driving, one for the elevator, and one for active cooling — all controlled via dual L298N H-bridges. The third battery powers a Raspberry Pi 4 through an adjustable voltage regulator configured to 5V/3A, which supports both the Pi and its connected USB camera. The Uno32 handles all control logic, sensor readings, and motor outputs.

Software System

Scoopy's logic is structured as a hierarchical state machine (HSM) implemented using the Event-Service Framework on the Uno32. Core behaviors include ArUco tag response, object pickup and dropoff, dynamic path planning, and jam recovery. The Raspberry Pi 4 is dedicated to ArUco tag detection using OpenCV and transmits the decoded information to the Uno32 via serial communication.
